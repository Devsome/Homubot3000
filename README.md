### Homubot3000
First but simple Discord Bot

#### Requirements
| Package | Version     |
| :------------- | :------------- |
| discord.js       | ^11.5.1       |
| dotenv-flow       | ^3.1.0       |
| moment       | ^2.24.0"       |

#### How To
* Clone this repository on your server or computer.
* run the command `npm install`
* copy the `.env.example` to `.env` and change the values
